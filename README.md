powderlines
===========

How to get up and running:
1. `gem install bundler`
2. `bundle install`
3. `npm install`
4. `grunt serve`
5. Go to http://localhost:4000/

Deploy:
1. Have a grunt-aws.json file in the project's root folder with our credentials.
2. `grunt deploy`
